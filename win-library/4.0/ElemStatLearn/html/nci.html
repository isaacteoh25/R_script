nci {ElemStatLearn}R Documentation

NCI microarray data (chap 14)

Description

Human Tumour Microarray Data

Usage

data(nci)

Format

The format is: num [1:6830, 1:64] 0.300 1.180 0.550 1.140 -0.265 ... - attr(*, "dimnames")=List of 2 ..$ : NULL ..$ : chr [1:64] "CNS" "CNS" "CNS" "RENAL" ...

Source

Originally from http://genome-www.stanford.edu/nci60/. Currently via the book's website, https://web.stanford.edu/~hastie/ElemStatLearn/datasets/nci.data.csv.

Examples

str(nci)
summary(nci)
